## 📌 About the Project

**DigiSmart Lahore** is a civics project website built to raise digital literacy and online safety awareness — specifically tailored for the people of **Lahore, Pakistan**. In an era where misinformation, cyberbullying, and online fraud are rampant, this platform equips students, parents, and citizens with the knowledge and tools to navigate the internet safely and responsibly.

The site is fully built in a single `HTML` file with embedded CSS and JavaScript — no frameworks, no dependencies, just clean and powerful web technology.

---

## ✨ Features

| Section | Description |
|---|---|
| 🏠 **Hero** | Animated landing with key statistics about Pakistan's digital landscape |
| 📖 **About** | Why digital literacy matters, with local Pakistan context and facts |
| ⚖️ **Digital Ethics** | Do's & Don'ts for responsible internet usage |
| 📰 **Fake News** | Step-by-step guide to spotting misinformation + red flags checklist |
| 😔 **Cyberbullying** | Types of cyberbullying, how to respond, and FIA contact information |
| 👨‍👩‍👧 **Parents' Guide** | Tabbed tips for parents on monitoring and supporting children online |
| 🧠 **Interactive Quiz** | 8-question quiz testing digital literacy knowledge with instant feedback |
| 🚨 **Report Cybercrime** | Step-by-step guide to reporting with emergency contact numbers |

## 🖼️ Design Highlights

- 🎨 **Dark navy + emerald green** color scheme with amber gold accents
- ✍️ **Playfair Display** serif headings paired with **DM Sans** body text
- 🌀 Animated background orbs, floating icons, and smooth scroll reveal effects
- 📱 Fully **responsive** — works on mobile, tablet, and desktop
- ⚡ **Zero dependencies** — everything runs from a single file

---

## 🧠 Quiz Topics Covered

The built-in quiz tests knowledge on real-world Pakistani digital scenarios:

1. Verifying viral WhatsApp forwards
2. Pakistan's **PECA 2016** cyber laws and penalties
3. What to do when a classmate is being cyberbullied
4. Role of the **FIA Cyber Crime Wing**
5. How **Two-Factor Authentication (2FA)** works
6. Identifying fake news URLs and red-flag headlines
7. Legal consequences of sharing private photos without consent
8. Creating strong, secure passwords
